# Motor Current Safety Monitoring System

## Description
This is a C++ application developed to monitor the safety condition of electric motors by comparing the rated current with the measured current.

## Features
- Monitor multiple motors
- Detect overload conditions
- Save reports
- View reports
- Clear reports
- About system page
- Menu-driven application

## C++ Concepts Used
- Classes and Objects
- Functions
- if...else
- switch
- for loop
- do...while loop
- File Handling
- Strings
- Input Validation

## How to Compile

```bash
g++ main.cpp -o motor
```

## How to Run

```bash
./motor
```

On Windows:

```powershell
.\motor.exe
```
